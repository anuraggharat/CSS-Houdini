CSS Houdini is an Umbrella term for a set of browser APIs that expose certain parts of the CSS rendering engine to the Developers.

This repository contains examples of CSS Houdini APIs

  1. Custom Properties and Values API
  2. Typed OM API
  3. Paint Worklet API - Example 1
  4. Paint Worklet API - Example 2
